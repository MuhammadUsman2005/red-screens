React Native UI Implementation

This project contains a React Native implementation of the provided Figma design. The focus is on building clean, pixel-accurate UI screens with proper structure and readable code.

The project is intended to be easy to understand, update, and extend if needed.

How to run the project locally:

Clone the repository
git clone <YOUR_GIT_URL>
cd <PROJECT_DIRECTORY>

Install dependencies
npm install

Run the project
npx react-native run-android
or
npx react-native run-ios

Project notes:

The UI follows the Figma design closely.

Screens are organized for clarity and scalability.

Any small technical adjustments (safe area, platform handling) are kept minimal.
